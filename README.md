# 🧑‍💻 Real-Time Collaborative Code Editor (Streamlit)

## 📌 Project Overview
The **Real-Time Collaborative Code Editor** is a web-based application that allows multiple users to write, edit, and share code in a common environment. This project is built using **Python** and **Streamlit**, providing a simple interface for collaborative coding.

---

## 🎯 Objectives
- Enable multiple users to collaborate on code  
- Provide a simple and interactive coding interface  
- Simulate real-time collaboration using shared storage  
- Understand synchronization concepts  

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit  
- **Backend:** Python  
- **Storage:** Text file (`shared_code.txt`)  

### Libraries Used
- streamlit  
- os  
- time  
- datetime
